## Operating Rule

All future independent agent development should follow:

```text
Draft -> Prompt -> Preflight -> Implementation -> Verification -> Human Review -> GitHub Commit/PR -> Retrospective -> AGENTS.md / Skills / Workflow Update
```

This is aligned with the Asperitas Agent Factory doctrine: AI-generated code is not trusted by default; it is verified, logged, reviewed, committed, and converted into reusable organizational process.
