# Asperitas AI Lead Control Plane v11.0

## Executive Bottom Line
This repository is the execution base for an Asperitas AI Lead / Biological Intelligence Factory control plane. It is not a generic chatbot repository. Its purpose is to convert biodiversity access into structured biological data, source-grounded intelligence, DBTL validation, IP/compliance trust, products/licensing, and global biological infrastructure.

## Critical Non-Confusion Boundary
This repository may contain prompts, source maps, registry scaffolds, RAG/KG/eval designs, or implementation guides. These are not proof that production vector DB, RAG, KG, eval suite, legal review, wet-lab validation, regulatory approval, or autonomous lab execution is complete. Claims must be upgraded only with implementation logs, tests, eval reports, approvals, and verified evidence.

## Minimum Phase Roadmap
Phase 0: source-grounded internal decision system - source registry, local corpus, RAG answer contract, citation eval, compliance tags, decision log.
Phase 1: guarded workflow automation and agent harness - fixed workflows, guarded tool calls, traces, targeted evals, human approvals.
Phase 2: biology KG and DBTL registry - entity resolution, pathway/species/protein/assay graph, experiment learning records.
Phase 3: biofoundry-ready operating system - LIMS/ELN integration, protocol versioning, sample tracking, model-assisted design under approval gates.

## Development Loop
Scope Lock -> Source & Risk Preflight -> Contract Design -> Minimal Implementation -> Eval Harness -> Dry Run & Regression -> Human Gate -> Merge & Evidence Log -> Learn Back.

## Repository Quality Gates
- Source registry before ingestion.
- License/confidentiality pass before embedding.
- Hybrid retrieval + rerank + top-K compression before answer synthesis.
- Claim verification record for investor/science/compliance claims.
- Compliance gate for CITES, Nagoya, LMO, biosafety, biosecurity, IP, privacy, export control.
- Trace/log/eval evidence before production claims.
- Human approval for high-risk biological, legal, regulatory, financial, or external communication outputs.
