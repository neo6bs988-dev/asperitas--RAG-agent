## Patterns to Absorb
- Industrialized biology depends on repeatable data generation, not one-off experiments.
- Model quality compounds when experimental data, metadata, and feedback loops are standardized.
- Data flywheel: more experiments -> more data -> better models -> better experiment selection -> more valuable data.

## Asperitas V1 Application
- Treat source registry and metadata schema as the earliest software equivalent of a biology data flywheel.
- Require every knowledge object to carry source, version, metadata, confidence, and verification state.
- Convert repeated research/ops workflows into skills and evals.
