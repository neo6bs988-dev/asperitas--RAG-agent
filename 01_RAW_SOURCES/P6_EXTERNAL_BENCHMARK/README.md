# 01_RAW_SOURCES/P6_EXTERNAL_BENCHMARK

Purpose: registered-only external benchmark source layer for founder/operator doctrine and AI-agent architecture references.

Status: governance scaffold only. No raw files, copyrighted pages, embeddings, vector DB records, or production RAG updates are created by this folder.

Source priority: P6 External Benchmark unless a specific source is a regulatory/government record, peer-reviewed/preprint scientific source, or approved internal source.

Use cases:
- Founder/operator doctrine comparison.
- AI-agent architecture benchmarking.
- External operating analogies for Asperitas Biological Intelligence Factory design.

Rules:
1. Register metadata before raw acquisition.
2. Review license, terms, IP, and compliance status before downloading or processing raw source text.
3. Preserve URL, organization, source type, source priority, disclosure, license status, retrieval date, and ingestion status.
4. Do not treat external benchmark doctrine as Asperitas fact.
5. Do not claim production ingestion or DB learning until raw acquisition, extraction, chunking, embeddings, retrieval, and eval logs exist.
