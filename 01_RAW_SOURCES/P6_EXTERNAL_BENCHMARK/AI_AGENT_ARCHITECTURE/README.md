Purpose: registered-only benchmark folder for AI-agent architecture frameworks, eval tooling, RAG systems, observability, and connector standards.

Status: no raw source files, vendored code, embeddings, or processed markdown are stored here yet.

- OpenAI Agents SDK, Agents API guide, Evals, and Cookbook.

Rules:
