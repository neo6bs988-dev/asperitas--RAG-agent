## 7. RAG / KG / Eval / Trace Control Plane
Minimum release gates:
- Source Registry: source_id, title, path/url, priority, confidentiality, license_status, collection_status, verification_status, owner, updated_at, allowed_use.
- Ingestion: loader, OCR/text extraction, dedup, chunking, metadata, license/confidentiality pass.
- Retrieval: hybrid search, metadata filters, rerank, top-K compression.
- Answer Contract: claim, source, evidence span, confidence, uncertainty, compliance tag.
- Knowledge Graph: species, genes, proteins, pathways, assays, projects, permits, claims.
- Eval Suite: retrieval, citation faithfulness, schema, regression, red-team, biosafety.
- Tracing: workflow, model, tool call, handoff, guardrail, latency, cost.

## 8. Prompt Lint Checklist
Before shipping any prompt/source/constitution update:
- Goal is explicit.
- Scope and negative scope are explicit.
- Evidence rules preserve source IDs and status.
- Constraints include biosafety, compliance, security, privacy, budget, token, CI limits.
- Output contract is fixed.
- Verification commands/evals are stated.
- Stop rules are stated.
- Non-overclaim boundary is present.
- Human approval gate is present for high-risk outputs.

## 9. Agent Module Boundaries
- Species Intelligence: taxonomy, distribution, permits, specimen metadata, conservation status. No commercial claim without provenance and legal status.
- Genome/Protein Discovery: sequence evidence, homologs, domains, variant hypotheses. No activity claim without assay or strong evidence label.
- Pathway/DBTL: DBTL planning and experiment learning records. No autonomous wet-lab execution.
- Literature/Patent: papers, patents, FTO flags, citation-backed opportunity map. No legal conclusion without counsel review.
- Compliance Gatekeeper: Nagoya, CITES, LMO, biosafety, privacy, license checks. Can block or escalate.
- COO Execution: roadmap, owners, bottlenecks, weekly cadence, investor artifacts. Must separate fact from plan.

## 10. Final Rule
When uncertain, state exactly what exists, what is inferred, what is pending, and what evidence would upgrade the claim.
